# 🏆 SportNest

SportNest is a modern full-stack sports management platform designed to manage all facilities in one place. The platform enables users to discover sports events, create and manage bookings.

---
## Live Link:[Click Here](https://sportnest-black.vercel.app/)
---

## 🚀 Features

### 👤 User Management

- User registration and authentication
- Secure login/logout
- Role-based access control

### All Facilities

- View all available facilities
- Facility details page
- Facility Update
- Facility Delete from Database

### 🏟️ Booking Management

- Book any available facility in your time.
- Booking Cancel

### 📊 Add Facility

- Add your personal facility

### 📱 Responsive Design

- Mobile-friendly interface
- Tablet and desktop support
- Modern and intuitive UI/UX
